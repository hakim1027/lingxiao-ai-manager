## 📖 使用教程

### 快速开始

```
1. 启动软件后自动读取本地 Cursor 账号
2. 在「使用统计」查看配额和消耗
3. 在「账号管理」切换不同账号
4. 需要时在「机器码管理」重置设备标识
```

### 常见问题

<details>
<summary><b>Q: 重置机器码会影响账号吗？</b></summary>

不会。机器码只是设备标识，重置后只需重新登录 Cursor 即可。
</details>

<details>
<summary><b>Q: 为什么显示 Token 过期？</b></summary>

长时间未使用 Cursor 可能导致 Token 过期，重新登录 Cursor 即可刷新。
</details>

<details>
<summary><b>Q: 支持 macOS 吗？</b></summary>

目前主要支持 Windows，macOS 版本正在开发中。
</details>
